# 八年级科学月考错题讲评网页幻灯片

这是 reveal.js 静态部署版，不依赖 node_modules，可直接用于 GitHub Pages。

## 本地预览

```bash
python3 -m http.server 8080
```

浏览器打开：

```text
http://127.0.0.1:8080
```

## 操作说明

- 方向键 / 空格：翻页
- F：全屏
- S：演讲者视图
- Esc：总览

## 导出 PDF

访问：

```text
https://你的用户名.github.io/仓库名/?print-pdf
```

然后用浏览器打印为 PDF。
